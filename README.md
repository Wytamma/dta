# PrioriTree DTA

## Installation

```bash
snk install wytamma/dta
```

## Usage

```bash
dta run \
    --input-trees data/delphy.trees \
    --prefix results/dta \
    --beast-chain-length 10000 \
    --beast-samples 100
```
